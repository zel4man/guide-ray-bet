# guide-ray-bet

## 简介

`guide-ray-bet` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为通用页面存储与展示的集合。

仓库中的 HTML 文件各自独立，互不依赖，可直接在浏览器中打开查看，也可部署到任意静态托管服务。

## 目录说明

```
guide-ray-bet/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
├── index.html      # 仓库首页或导航页
├── README.md       # 本文件
└── LICENSE         # 许可证文件（如有）
```

- **pages/**：每个 HTML 文件对应一个独立页面，文件名建议使用英文或拼音，避免特殊字符。
- **assets/**：页面所需的样式、脚本、字体、图片等资源按需存放，保持目录整洁。
- **index.html**：可提供简单的页面索引或导航，方便浏览。

## 页面归档说明

本仓库中的页面可能来源于不同时期、不同用途的独立项目。每个页面均保持原始结构，未做额外修改。页面之间无关联，可单独查看或引用。

归档目的仅为记录和展示，不提供持续更新或功能维护。页面内容可能包含静态演示、信息展示或实验性功能，请自行判断使用场景。
